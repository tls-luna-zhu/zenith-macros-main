Features

Built for Competitive Play

External by design. Zenith Macros runs as a separate Windows process and never injects into Minecraft or touches its memory, packets, or files.

Fully External
Runs outside Minecraft’s process. No injection, no memory reads, no packet tampering — clean external automation.
Focus Lock
Macros suspend the moment Minecraft loses focus and resume instantly when you return — no accidental input in other apps.
Low-Latency Engine
Optimized scheduling and lightweight loops keep input timing tight while reducing CPU overhead on long sessions.
Triggerbot
Pixel‑accurate crosshair detection with Normal, Smart Crit, and S‑Tap modes for controlled, repeatable hits.
Profiles
Save and switch full configurations instantly — different servers, playstyles, or keybind layouts.
Stealth Mode
One hotkey hides the window and taskbar entry while macros keep running.

FAQ

Common Questions

Everything you need to know before getting started.

How does Triggerbot decide when to fire?
Triggerbot reads only the center crosshair state and reacts to valid red/blue target states. It does not use edge-box colors, and cooldown logic is tuned for consistent first-hit response.
What versions of Minecraft are supported?
Supports Minecraft Java across major launchers: Vanilla, Lunar, Badlion, Feather, and PVP Lounge. Client switching doesn’t require reconfiguration.
Do I need to run as Administrator?
No. Zenith Macros runs as a standard user process. It works best when Minecraft is also running as a standard user, which is the default for major launchers.
How does the license system work?
Keys are delivered instantly by email. Each key activates on one machine (hardware‑bound, not IP‑bound). Monthly renews automatically until cancelled; Lifetime is one-time and does not renew.
What is Focus Lock and when should I disable it?
Focus Lock pauses macros whenever Minecraft is not the active window to prevent accidental input elsewhere. Disable only if you intend to run macros in the background.
