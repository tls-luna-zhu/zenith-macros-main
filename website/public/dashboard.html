How the EXE Builder works
1
Filename
Controls what the EXE is called in your Downloads folder and on your Desktop. Set it to Spotify and the file saves as Spotify.exe.
2
Display Name, Company & Description
Shown in Task Manager (Description column), File Properties (right-click → Properties → Details), and the Start Menu search result. All written into the EXE's version resource at build time.
3
Icon
Selecting a disguise preset replaces the icon embedded in the EXE — so File Explorer, the Taskbar, and Alt+Tab all show the preset colour instead of the Zenith icon.
The app itself is unchanged
Only the metadata & icon resources are patched. Every macro, keybind, and feature works identically.
